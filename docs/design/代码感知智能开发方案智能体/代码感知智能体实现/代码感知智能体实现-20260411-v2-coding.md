## 1. v2 核心变更（与 v1 差异）

| 变更点 | v1 | v2 |

| ScanNode 执行方式 | 调 4 Tool → LLM 综合 → Profile JSON | 缓存命中读文件 → 未命中走 ProfileGeneratorChain |

| Profile 格式 | JSON（7 维度） | Markdown（10 维度） |
| State 字段 | `structure` + `topology`（JSON） | `projectProfile: String`（Markdown 原文） |
| CodeAwarenessAgent | 必须 | 废弃 |
| ProfileIndexTool 输入 | Profile JSON | Markdown 按 `## ` 分片 |
| 配置化 | 无 | `devplan.profile.generators.*` 开关控制 |

## 2. 核心业务规则

保留 v1 全部规则，变更和新增：

- **R6 修改**：Profile 维度从 7 扩展为 10（新增 SERVICE_CAPABILITY / EXTERNAL_DEPENDENCY / CODING_CONVENTION）
- **R7（新增）**：Markdown 缓存优先 — 优先读 `docs/project-profile.md`，过期或缺失走 SPI 链
- **R8（新增）**：SPI 链优先外部工具 — ClaudeCode(order=10) → Fallback(order=99)
- **R9（新增）**：生成器可配置开关 — 每个生成器独立 `enabled` 配置项
- **R10（新增）**：可强制要求 LLM 生成器 — `require-llm-generator=true` 时跳过非 LLM 生成器（InternalFallback），全部失败直接报错不兜底

## 6. ProfileGenerator SPI 伪代码

```java
// Domain 层接口
public interface ProfileGenerator {
    Optional<String> generate(String projectPath);
    boolean isAvailable();
    int order();
    /** 是否基于大模型 CLI 编程工具。require-llm-generator=true 时只有此类生成器参与链。 */
    default boolean isLlmBased() { return false; }
}

// 编排器
@Service
public class ProfileGeneratorChain {
    private final List<ProfileGenerator> generators; // 按 order 排序注入
    private final DevPlanProfileProperties properties;

    public String generateOrThrow(String projectPath) {
        boolean requireLlm = properties.isRequireLlmGenerator();

        for (ProfileGenerator gen : generators) {
            if (requireLlm && !gen.isLlmBased()) continue;  // 强制模式跳过非 LLM 生成器
            if (!gen.isAvailable()) continue;
            Optional<String> result = gen.generate(projectPath);
            if (result.isPresent()) return result.get();
        }

        String msg = requireLlm
            ? "All LLM-based generators failed (internal fallback disabled): "
            : "All generators failed: ";
        throw new ProfileGenerationException(msg + projectPath);
    }
}
```

## 7. ClaudeCodeProfileGenerator 伪代码

```java
@Component
@ConditionalOnProperty(prefix = "devplan.profile.generators.claude-code", name = "enabled", havingValue = "true")
public class ClaudeCodeProfileGenerator implements ProfileGenerator {

    @Value("${devplan.profile.generators.claude-code.cli-path:claude}")
    private String cliPath;

    @Value("${devplan.profile.generators.claude-code.timeout-seconds:300}")
    private int timeoutSeconds;

    @Override
    public Optional<String> generate(String projectPath) {
        try {
            Path profilePath = Path.of(projectPath, "docs", "project-profile.md");

            ProcessBuilder pb = new ProcessBuilder(
                cliPath,
                "--print",
                "--dangerously-skip-permissions",
                "--output-format", "text",
                "请执行 generate-project-profile skill，分析项目并生成画像文档。项目路径：" + projectPath
            );
            pb.directory(new File(projectPath));
            pb.redirectErrorStream(true);

            Process process = pb.start();
            boolean finished = process.waitFor(timeoutSeconds, TimeUnit.SECONDS);
            if (!finished) { process.destroyForcibly(); return Optional.empty(); }
            if (process.exitValue() != 0) { return Optional.empty(); }

            if (Files.exists(profilePath)) {
                return Optional.of(Files.readString(profilePath));
            }
            return Optional.empty();
        } catch (Exception e) {
            log.error("ClaudeCodeProfileGenerator failed", e);
            return Optional.empty();
        }
    }

    @Override
    public boolean isAvailable() {
        try {
            Process p = new ProcessBuilder(cliPath, "--version").start();
            return p.waitFor(5, TimeUnit.SECONDS) && p.exitValue() == 0;
        } catch (Exception e) { return false; }
    }

    @Override
    public int order() { return 10; }

    @Override
    public boolean isLlmBased() { return true; }
}
```

**CLI 参数说明：**
- `--print`：非交互模式
- `--dangerously-skip-permissions`：跳过权限确认（服务端无人值守场景）
- `--output-format text`：纯文本输出
- 工作目录 = `projectPath`：确保文件路径正确

## 8. ScanNode 伪代码

```java
public DevPlanState execute(DevPlanState state) {
    Path profilePath = Path.of(state.projectPath(), "docs", "project-profile.md");
    String markdown;

    if (isProfileFresh(profilePath, state.projectPath())) {
        markdown = markdownReader.readFull(profilePath);
    } else {
        markdown = generatorChain.generateOrThrow(state.projectPath());
        // ClaudeCode 自己写文件，Fallback 需手动写
        if (!Files.exists(profilePath)) {
            Files.createDirectories(profilePath.getParent());
            Files.writeString(profilePath, markdown);
        }
    }

    asyncExecutor.submit(() -> profileIndexTool.indexFromMarkdown(markdown, state.projectPath()));

    return state.withProjectProfile(markdown).withStatus("SCANNING_COMPLETE");
}

private boolean isProfileFresh(Path profilePath, String projectPath) {
    if (!Files.exists(profilePath)) return false;
    try {
        Instant profileTime = Files.getLastModifiedTime(profilePath).toInstant();
        Instant lastCommit = getLastCommitTime(projectPath);
        return profileTime.isAfter(lastCommit);
    } catch (Exception e) { return false; }
}
```

## 9. ProfileMarkdownReader 伪代码

```java
public Map<ProfileDimension, String> parse(String markdown) {
    Map<ProfileDimension, String> result = new LinkedHashMap<>();
    String[] sections = markdown.split("(?=^## )", Pattern.MULTILINE);

    for (String section : sections) {
        Matcher m = Pattern.compile("^## (\\d+)\\.\\s+(.+)$", Pattern.MULTILINE).matcher(section);
        if (m.find()) {
            int index = Integer.parseInt(m.group(1));
            ProfileDimension dim = ProfileDimension.fromIndex(index);
            if (dim != null) {
                result.put(dim, section.trim());
            }
        }
    }
    return result;
}
```

## 13. 异常处理要点

| 场景 | 处理 |
|------|------|
| profile.md 不存在或已过期 | ProfileGeneratorChain 按优先级尝试 |
| Claude Code CLI 不存在 | `isAvailable()=false`，跳过，尝试 Fallback |
| Claude Code 执行超时 | `destroyForcibly()` + empty，尝试 Fallback |
| Claude Code 成功但未生成文件 | empty，尝试 Fallback |
| 所有生成器均失败 | 抛 ProfileGenerationException，ScanNode 标记 FAILED |
| `require-llm-generator=true` 且 LLM 生成器全部失败 | 直接报错，**不回退 Fallback**，ScanNode 标记 FAILED |
| profile.md 格式异常 | 视为过期，重走 SPI 链 |
| projectPath 不存在 | FAILED + error msg |
| Qdrant 不可用 | 画像不受影响，indexStatus = DEGRADED |
| git 不可用 | 视为过期，走 SPI 链 |
