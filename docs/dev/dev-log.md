## 记录规范

每次代码变更后，追加到对应周的 `dev-log-YYYY-WNN.md` 文件末尾，包含：
- 日期
- 任务描述
- 变更文件
- 关键设计决策

若当周文件不存在，新建 `dev-log-YYYY-WNN.md`，并在本索引中补一行。
周次使用 ISO 8601 标准（Python: `date.isocalendar()`）。
